## 📌 이 시스템은 무엇인가요?

공장 현장에서 **터치스크린으로 조작**하는 세탁물 입출고 관리 앱입니다.

기존에 수기나 엑셀로 관리하던 아래 업무들을 화면에서 바로 처리할 수 있습니다.

| 기능 | 설명 |
|------|------|
| 세탁물 관리 | 거래처별 세탁 완료 수량, 불량 수량, 재고를 실시간으로 입력하고 확인 |
| 출고 신청 | 세탁이 끝난 품목을 거래처에 출고할 때 수량을 선택하고 등록 |
| 불량 처리 | 불량으로 분류된 품목을 폐기 / 반환 / 기타로 처리 |
| 출고 현황 | 오늘, 이번 주, 이번 달 출고 내역을 한눈에 확인하고 수정 |

---

## 👥 사용 대상

- **현장 직원** — 터치스크린에서 세탁 수량 입력, 불량 처리, 출고 신청
- **관리자** — 출고 현황 조회, 내역 수정, 전표 재출력
- **대표 / 기획자** — 거래처별 출고 실적 및 재고 현황 파악

---

## 🗂️ 거래처 및 품목 구성

### 거래처 유형

🏨 호텔 &nbsp;&nbsp; 🏡 펜션 &nbsp;&nbsp; 🌴 리조트 &nbsp;&nbsp; 🏢 기타

### 품목 카테고리

| 카테고리 | 품목 |
|----------|------|
| 🏷️ 타올 | 대타올, 중타올, 소타올, 목욕가운, 슬리퍼타올 |
| 🛏️ 시트 | 시트S, 시트D, 시트Q, 시트K, 두베커버S/D/K, 베개커버 |
| 👔 유니폼 | 상의, 하의, 앞치마, 조끼, 모자 |

## 🔄 업무 흐름

### 1. 세탁물 수량 입력

```
거래처 선택 → 카테고리 선택 (전체/타올/시트/유니폼)
→ 품목 터치 → 세탁완료 또는 불량 수량 입력
→ 수량 적용 → 재고 자동 계산
```

> 💡 재고는 **세탁완료 + 불량** 으로 자동 계산됩니다. 별도 입력 불필요.

---

### 2. 출고 신청

```
거래처 선택 → 출고할 품목 터치 → 수량 입력
→ 출고 목록 확인 → 출고 일시 설정 → 출고 확인
```

> 💡 출고 확인 시 세탁완료 수량에서 자동 차감되고 출고 기록에 저장됩니다.

---

### 3. 불량 처리

```
거래처 선택 → 불량 품목 터치
→ 처리 방식 선택 (전체 처리 / 부분 처리)
→ 처리 유형 선택 (폐기 / 반환 / 기타)
→ 처리 실행 → 재고 자동 재계산
```

---

### 4. 출고 현황 확인 및 수정

```
날짜 필터 선택 (오늘 / 이번 주 / 이번 달 / 전체)
→ 거래처 필터 (선택 사항)
→ 출고 내역 목록 확인
→ 수정 필요 시: 행 터치 → 수량 또는 일시 수정 → 저장
→ 전표 재출력 가능
```

## 🖥️ 화면 구성 — 테마 A vs 테마 B

현재 두 가지 화면 디자인(테마)을 제공하며 홈 화면에서 선택할 수 있습니다.

### 테마 A — 대시보드형

| 특징 | 내용 |
|------|------|
| 레이아웃 | 상단 탭 메뉴 + 카드 그리드 방식 |
| 색상 | 딥 인디고(남색 계열) + 청록 조합 |
| 품목 표시 | 카드 형태로 수치를 크게 표시 |
| 수량 입력 | 우측 입력 패널에서 숫자 키패드로 입력 |

**화면 구성**

```
┌─────────────────────────────────────────────────┐
│  🏭 세탁물관리  출고신청  불량처리  출고현황        │  ← 상단 탭 (크고 터치하기 편한 버튼)
├──────────┬─────────────────────────┬────────────┤
│          │  [ 대타올 카드 ]         │            │
│  거래처   │   세탁완료  불량  재고    │  수량 입력  │
│  목록    │     48    2    50       │  패널      │
│          │  [ 중타올 카드 ]         │            │
│          │   세탁완료  불량  재고    │            │
│          │     35    0    35       │            │
└──────────┴─────────────────────────┴────────────┘
```

---

### 테마 B — 사이드 패널형

| 특징 | 내용 |
|------|------|
| 레이아웃 | 좌측 아이콘 네비 + 리스트 행 방식 |
| 색상 | 스카이 블루 + 슬레이트 조합 |
| 품목 표시 | 가로 행(테이블) 형태 |
| 수량 입력 | 우측 패널에서 키패드 입력 |

**화면 구성**

```
┌────┬──────────┬──────┬────────────────────────────┬──────────┐
│    │          │      │  품목명   세탁완료  불량  재고 │          │
│ 네  │  거래처   │ 구분  │  대타올    48     2    50   │  수량    │
│ 비  │  패널    │ 탭   │  중타올    35     0    35   │  입력    │
│    │          │      │  소타올    22     1    23   │  패널    │
│    │          │      │  목욕가운  18     0    18   │          │
└────┴──────────┴──────┴────────────────────────────┴──────────┘
```

## ✅ 이번에 작업한 주요 내용

### 테마 A 전면 개편

- **색상 교체**: 촌스러운 주황색 → 세련된 딥 인디고(남색) + 청록 조합으로 전면 변경
- **글자 크기 전체 확대**: 공장 현장에서 멀리서도 잘 보이게 전체적으로 키움
- **터치 버튼 크기 확대**: 상단 탭, 거래처 목록, 카테고리 탭, 적용 버튼 모두 크게 개선
- **불량 뱃지 제거**: 거래처 목록 옆에 표시되던 불량 수량 뱃지 제거 (불량처리 페이지에서 확인)
- **출고현황 수정 개선**: 행을 터치하면 바로 수정 패널이 열리도록 변경
- **전표 재출력 버튼 추가**: 출고현황 수정 패널에 전표 재출력 버튼 신규 추가
- **오늘/이번주/이번달 버튼**: 훨씬 크고 터치하기 편하게 개선, 서로 간격도 넓힘

### 테마 B 개선

- **선택 방식 정리**:
  - 세탁물 관리 → 한 번에 하나씩 선택 (단일 선택)
  - 출고 신청 → 여러 품목 동시 선택 유지 (멀티 선택)
  - 불량 처리 → 한 번에 하나씩 선택 (단일 선택)
- **거래처 패널 확대**: 글자와 박스 크기를 키워 터치하기 편하게 개선
- **중앙 품목 리스트 비율 조정**: 가운데가 너무 넓었던 것을 줄이고, 좌우 패널을 더 넓게 조정
- **수치 글자 크기 확대**: 세탁완료, 불량, 재고 숫자 전체적으로 크게 개선
- **불량 처리 버튼 개선**: 전체처리/부분처리 버튼을 더 크고 명확하게 변경
- **출고현황 수정 개선**: 행 터치로 수정 패널 오픈, 전표 재출력 버튼 추가
- **오늘/이번주/이번달 버튼**: 크기와 간격 대폭 확대

## 📦 재고 계산 방식

재고는 별도로 입력하지 않아도 **자동으로 계산**됩니다.

```
재고 = 세탁완료 + 불량
```

| 상황 | 변화 |
|------|------|
| 세탁완료 수량 입력 | 재고 자동 업데이트 |
| 불량 수량 입력 | 재고 자동 업데이트 |
| 출고 확인 | 세탁완료 감소 → 재고 자동 감소 |
| 불량 처리 완료 | 불량 감소 → 재고 자동 감소 |
